# Date-Simulator
It is designed as a beginner-friendly project demonstrating structured programming, user interaction, and simple decision-based flow control using only the C++ standard library.

The game focuses on a light narrative experience centered around building a connection between two characters. Through a sequence of choices, the player influences emotional progression represented by abstract metrics such as love, heart, and date success rating. The system is intentionally simple and does not rely on any external libraries.

Each decision affects the outcome differently, allowing multiple endings based on player behavior. Positive interactions increase emotional alignment, while poor choices may reduce compatibility. Randomness is also included to simulate unpredictable conversation outcomes.

The purpose of this repository is educational, showing how even small programs can create engaging interactive logic using basic C++ tools such as loops, conditionals, functions, and random number generation. Developers can extend the system by adding dialogue trees, multiple characters, or branching storylines.
